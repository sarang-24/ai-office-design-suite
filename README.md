# AI Office Design Suite 🏢📐🎨

An advanced, AI-powered interactive dashboard application designed to streamline modern office planning, furniture recommendation, analytics, CRM management, and team collaboration.

Built with a state-of-the-art frontend tech stack, this application provides an immersive UI/UX for architects, project managers, and clients to design, visualize, and budget office space renovations.

---

## ✨ Features

- **📊 Analytics Dashboard**: Real-time business reporting, ROI calculations, and project performance trackers using Recharts.
- **🛋️ Furniture Recommender**: AI-assisted recommendation engine suggesting office furniture layouts, styles, and cost-efficient fits.
- **🎨 Space Planning & Design Visualizer**: Interactive layout tool and 3D virtual tour module to inspect design spaces in real time.
- **🤝 Project Management & CRM**: Gantt schedules, calendar tasks, lead tracking, and collaboration panels for project squads.
- **💰 Quotation & Billing Suite**: On-the-fly quotation generator, billing trackers, and professional invoicing systems.
- **🔐 Lock Screen**: Secure dashboard access guard keeping operational client data private.

---

## 🛠️ Tech Stack

- **Framework**: React 18 & Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 & Radix UI
- **Animations**: Framer Motion
- **Charts & Data**: Recharts
- **Icons**: Lucide React & Material-UI Icons

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sarang-24/ai-office-design-suite.git
   cd ai-office-design-suite
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
